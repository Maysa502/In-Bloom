# In Bloom

![In Bloom Banner](In-Bloom.Banner.png)

## Descrição
"In Bloom" é um projeto de site para uma loja online que vende souvenirs da banda Nirvana. O site é desenvolvido utilizando HTML, CSS e JavaScript para proporcionar uma experiência interativa e responsiva aos usuários.

## Estrutura do Projeto
O projeto está organizado da seguinte forma:

- **index.html**: Página inicial do site.
- **produtos.html**: Página de exibição e compra dos produtos.
- **contatos.html**: Página de contato para interação com os clientes.
- **sobre.html**: Página com informações sobre a loja e a banda Nirvana.

## Recursos Utilizados
- **HTML**: Estruturação das páginas.
- **CSS**: Estilização e design das páginas.
- **JavaScript**: Funcionalidades interativas e dinâmicas.
